# Age Calculator Using HTML, CSS & JavaScript

## Project Name

**Smart Age Calculator**

## Description

Smart Age Calculator is a simple web-based application built using **HTML**, **CSS**, and **JavaScript**. It allows users to calculate their exact age based on their **date of birth**. The calculator displays the age in **years, months, and days**, making it useful and easy to understand.

## Features

* Calculate age from date of birth
* Accurate results in years, months, and days
* Simple and clean user interface
* Fully responsive design
* Built with pure HTML, CSS, and JavaScript

## Technologies Used

* **HTML** – Structure and input fields
* **CSS** – Styling and layout
* **JavaScript** – Age calculation logic

## How It Works

The user enters their date of birth. JavaScript uses the `Date` object to get the current date and calculates the difference between the current date and the entered date of birth. The result is then displayed as the user’s age.

## How to Use

1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Enter your date of birth.
4. Click the calculate button to see your age.

## Customization

* Change layout and colors in `style.css`.
* Modify calculation logic in `script.js`.
* You can add features like next birthday countdown.
